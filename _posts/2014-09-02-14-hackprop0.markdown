## Dance Pad Controller Refurbish
I've recently acquired an old Dance Dance
Revolution dance-pad controller. My plan for
this hack is to interface it with an Arduino
so it can be used as a virtual keyboard.

## Prerequisites:

* PySerial (pip module)

* Arduino
