Which project did your team choose?

MonoGame

2-liner description of the project?

MonoGame is a port of Microsoft's XNA Game Framework to the Mono language. MonoGame takes the features provided by XNA and allows for cross-platform deployment.

What will each team member's role be?

Believable answer, something that might go as a title on a resume (emphasis on believable.)

Source Code Repository URL?

https://github.com/mono/MonoGame

What are the easy parts?

  • MonoGame is a very active project on GitHub (link provided above)
  • MonoGame has a somewhat active forum on codeplex.com ( monogame.codeplex.com/discussions )
  • The MonoGame Core Dev Team is very active on Twitter ( https://twitter.com/MonoGameTeam )

What are the hard parts?

  • MonoGame's IRC is only somewhat active, this will provide a slight challenge when trying to communicate with the core dev team.
  • While MonoGame is an active project it has a long-term development cycle. It's last release was over a year ago.
